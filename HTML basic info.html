HTML Cheat Sheet

This cheat sheet or HTML code quick reference is a list of the most common HTML tags and their attributions.
There are also relevant sections in an easy-to-read format placed down below.

How to begin?

At the top of your HTML document, you should include a doctype. This lets the script know about which language the code is actually about.
When you begin with HTML code, the first thing you want to do is include 'html' and close it at the end.
After that, you'll place a head and a body. Inside of 'head' a title can be placed.
Inside 'body' are 90% of the things placed, the content itself.
